lbs，一个基于lbs（基于位置的服务）的社交小平台，比较麻烦的地方应该是高德地图那一块，客户端跟后台组件我杂糅在一起了，之后会把它们分开吧（多入口）。还有很多优化的地方！

### Clone
```
git clone git@github.com:476542084/Lbs.git
```
### 启动

>启动暂时请求原有java后台（暂时在别的同学那里，之后我会重写一个koa.js版本后台，敬请期待...）
贴上<a href="https://github.com/476542084/Lbs/blob/master/package.json" >API说明</a>

## Install
```
npm install
```

### Build
```
npm run build
```


### 相关技术栈：

+ vue + vuex + vue-router + vant + element + echart
+ token控制
+ 后台单点登录机制
+ ....

+ 客户端跟后台分开，多入口

+ 附近人社交（摇一摇？）
+ 小程序版
+ ....
